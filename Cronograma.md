# Tópicos em Desenvolvimento de Aplicativos

## Aula 1 – Introdução ao Desenvolvimento Mobile e React Native

**Conceitos**

* Panorama do desenvolvimento mobile (nativo vs híbrido vs cross-platform)
* Arquitetura do React Native
* Node.js e NPM/Yarn
* Expo vs React Native CLI

**Prática**

* Instalação do ambiente
* Criação do primeiro projeto
* Estrutura de pastas
* Executando no emulador/dispositivo físico

**Conceitos**

**Prática**

* Criar uma tela simples com estado

# 🎯 Sugestão de Projeto Integrador

Os alunos podem desenvolver ao longo do semestre:

* 📱 App de marketplace
* 📱 App de controle financeiro
* 📱 App de rede social simples
* 📱 App de tarefas com autenticação

E a cada aula incorporar o novo conceito ao projeto.


Se você quiser, posso:

* Transformar isso em **plano de ensino formal (ementa + objetivos + avaliação)**
* Criar **rubrica de avaliação dos projetos**
* Sugerir **trabalhos e provas**
* Montar versão com carga horária detalhada (2h, 3h ou 4h por encontro)**
